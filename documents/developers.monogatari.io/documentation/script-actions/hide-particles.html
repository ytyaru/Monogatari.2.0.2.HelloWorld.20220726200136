Hide Particles
Hide a particle system animation

Description

1
'hide particles'
Copied!
This hide action will stop the currently shown particle system. To learn more about the particle systems, read the Show Particles documentation.
Action ID: Particles::Hide
Reversible: Yes
Requires User Interaction: No

Parameters

None
